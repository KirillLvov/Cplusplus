This is a project, which realizes the class Process.
Process can be created or be closed, its stdin can be closed. 
One can read data from the Process through the method read() or read a particular number of bytes through the method readExact().
One can write to the Process using the method write() or write a particular number of bytes using the method writeExact().
An example of usage of the Process class is in the src/main.cpp.
